
# Barnes Pharmaceutical - Vite + React (Tailwind)

## Quick start (locally)
1. Install dependencies:
```
npm install
```
2. Run dev server:
```
npm run dev
```
3. Build for production:
```
npm run build
```
4. Preview production build:
```
npm run preview
```
